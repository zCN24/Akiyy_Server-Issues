# Akiyy Minecraft Server 反馈中心

这里是专门用来收集大家游玩时的**问题、崩溃、建议、想加的mod、平衡调整**的地方～  
不在这里发，容易被刷屏淹没哦！

## 如何提交反馈

1. 点击右上角绿色的 **New issue** 按钮  
2. 选择下面合适的模板（如果有）或者直接用空白的  
3. 写清楚标题 + 描述 → Submit new issue

## 建议的反馈内容

**崩溃/报错**  
- 标题：【崩溃】xxxxxxxxxx  
- 内容：发生了什么 → 什么时候 → 单人/多人 → 日志最后20行（用 ``` 包起来）→ 截图（如果有）

**卡顿/延迟**  
- 标题：【性能】主世界/某个维度卡顿严重  
- 内容：TPS多少？哪个mod/区块？在线人数？截图F3界面

**平衡/建议**  
- 标题：【建议】XXX物品/机制太强/太弱  
- 内容：具体描述 → 你希望改成什么样

**想加的mod/数据包**  
- 标题：【模组请求】XXX（mod名字）  
- 内容：mod链接 + 为什么想加 + 大概影响

## 已有规则 / 注意事项
- 请先搜索是否已有相同issue（避免重复）
- 不要在这里吵架/开地图炮，文明发言
- 严重作弊/破坏直接截图+坐标私我，不用在这里发
- 好的建议会被优先考虑, 感谢每一条反馈！
