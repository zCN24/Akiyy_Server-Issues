# 枫林月语整合包 v2.0.0 更新日志

**发布日期**：2026-02-03
**Minecraft 版本**：1.21.1
**Loader**：Fabric
**客户端更新类型**：轻量优化 / 性能向 / 兼容 Akiyy Server
**更新标签**：release

## 本次主要更新亮点

1. **极致精简整合包体积**
   - 极致缩小整合包下载体积（不足1Mb）
   - 所有资源文件、部分 mod 文件从包体内移除
   - 玩家第一次进服时会自动从可靠源下载所需内容  
   - 大幅降低初次下载大小，提升分发效率

3. **游戏性能全面优化**  
   - 降低整体内存占用  
   - 优化内存管理逻辑，减少长时间游玩后的卡顿与掉帧

4. **修复关键崩溃问题**  
   - 针对部分 mod 导致的内存泄露问题进行修复  
   - 显著减少因内存溢出引发的游戏崩溃  
   - 已测试稳定性大幅提升

5. **优化与 Akiyy 服务器的协同性**  
   - 调整部分 mod 配置与 Akiyy 服务器环境更匹配  
   - 减少跨服/联机时的兼容性问题  
   - 提升在 Akiyy 服务器上的实际游玩体验
